ArDrive Mobile is just as intuitive as the ArDrive Web because it shares [the same codebase](https://github.com/ardriveapp/ardrive-web). All of the buttons that exist on ArDrive web are used in the mobile & share the same functions. The main difference is the ‘+' button acting as another 'NEW’ button in the web app.

![](https://help.ardrive.io/hc/article_attachments/9947906398235/file)

Check out the [ArDrive Explorer View](https://ar-io.zendesk.com/hc/en-us/articles/7782574688667-ArDrive-Explorer-View) for what each of the buttons do when selecting a Drive, Folder, or File.
