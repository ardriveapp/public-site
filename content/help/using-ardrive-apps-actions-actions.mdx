After your data is uploaded and secured on the Arweave network, ArDrive lets you perform various actions to edit, share and download your content.

**Drives:** When in a drive you can click on the “Show Menu” Button

![](https://help.ardrive.io/hc/article_attachments/15405246341915)

This will bring up the dropdown:

![](https://help.ardrive.io/hc/article_attachments/15405246417179)

You will now have your choice to Rename, Share, Export the Drive Contents or see the details (More info) of the Drive.

**Folder**: Clicking on a folder will bring up the details bar to the right and the ability to Rename or Move.

![](https://help.ardrive.io/hc/article_attachments/15405224649115)

**Files**: Clicking on a file will bring up the details bar and the ability to:

-   Share the file
    
-   Download the file
    
-   Preview the file (will show you the permanent URL given to the file in your web browser)
    
    -   **Note:** Only for Public files and not Private Files
        
-   Rename the file
    
-   Move the file
    

![](https://help.ardrive.io/hc/article_attachments/15405246612763)

For a more in-depth guide of all these functions see below:

-   [Download a file](/help/using-ardrive-apps-actions-download-a-file/)
    
-   [Share a file](/help/using-ardrive-apps-actions-file-sharing/) or [drive](/help/using-ardrive-apps-actions-drive-sharing/)
    
-   [Preview](/help/using-ardrive-apps-actions-public-preview/) a public file
    
-   [Edit](/help/using-ardrive-apps-actions-rename-a-file-or-folder/) a file or folder name
    
-   [Move](/help/using-ardrive-apps-actions-move-a-file-or-folder/) a file or folder name
