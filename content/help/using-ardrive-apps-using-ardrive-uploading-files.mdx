After [creating a Drive](/help/using-ardrive-apps-using-ardrive-create-new-drive/) you are ready to upload a file.

Click into the Drive where you want to upload your file.

Now click on the **+NEW** button and select “Upload file(s)”

ArDrive will now allow you to select from the file(s) on your desktop.

Select which file you want and press “Upload”

ArDrive will then calculate the cost of the file(s) you want to upload to the permaweb. A pop up screen will appear that will confirm the file(s) you have selected, the size of each file, and the one-time cost.

You will now have the ability to cancel the transaction at no cost or upload the file for the cost indicated.

## What file sizes can I upload?

Upload limits depend on upload method, browser, and platform. See [ArDrive App Limits](/help/faqs-help-and-troubleshooting-guides-ardrive-app-limits) for the current limits.

Our calculation of file size may be slightly larger than the file size listed on your computer given the unique way that data is stored and encrypted on the permaweb.

## How long does it take to upload?

Decentralized, permanent data storage doesn’t happen in an instant. Upload speeds are dependent on your internet connection. ArDrive waits til there are 15 confirmations on your uploaded data to ensure there are no forks in the chain.

**NOTE:** Do not leave from the uploading screen until the upload completes. If you exit the browser before it has completed your file will not be uploaded. You will not be charged if a file does not fully upload, but any progress towards uploading your file will be lost.

## Understanding File Status

ArDrive tries to track the status of each transaction in order to let you know when your files are stored on the network.

-   Green status means the file was successfully replicated and mined on the network.

-   Orange status means the file upload transaction is in progress.

    -   It could take up to 30 minutes for this process to complete, permanence requires patience!

-   Red status means the file upload transaction may have failed

    -   If a file is not mined, the AR payment made for this file is refunded.
