[turbo-gateway.com](http://turbo-gateway.com/ "http://turbo-gateway.com") is the domain of the major Arweave ecosystem gateway that is used to access to the Permaweb. By default, [app.ardrive.io](http://app.ardrive.io/ "http://app.ardrive.io") connects to this gateway for uploading and downloading data and querying its index for drive, file and folder information.

Changing the gateway may be useful in the cases when you can no longer access the default gateway (for example due to an ISP block) or want to connect with another decentralized gateway for Permaweb services.

Switching the gateway to [ar-io.net](http://ar-io.net/ "http://ar-io.net"), an alias gateway which has 100% parity with [turbo-gateway.com](http://turbo-gateway.com/ "http://turbo-gateway.com"), can be done in five simple steps in the ArDrive App from the initial login screen.

1.  Open the ArDrive web app in your browser - app.ardrive.io
    
2.  Right click & select `Inspect`
    
    ![Screenshot\_2023-01-10\_at\_12.17.39\_PM.png](https://help.ardrive.io/hc/article_attachments/12000464226843)![](blob:https://ardrive.atlassian.net/8b03175c-642b-477c-9db8-051bfc96aeb1#media-blob-url=true&id=a3c54116-585a-4d0c-b800-f9b4ed0349da&collection=contentId-635174929&contextId=635174929&height=1008&width=1792&alt=)
    
3.  Select `Console` to access the browser's JavaScript console
    
    ![](blob:https://ardrive.atlassian.net/4d3f92a5-d4b6-4c4a-a0b4-3406086d9737#media-blob-url=true&id=b957837b-1f09-48ad-9b8b-d8634447cf29&collection=contentId-635174929&contextId=635174929&height=1008&width=1792&alt=)![Screenshot\_2023-01-10\_at\_12.19.18\_PM.png](https://help.ardrive.io/hc/article_attachments/12000465354011)
    
4.  Copy & paste the following example next to the blue `>` to set a [ar-io.net](http://ar-io.net/ "http://ar-io.net") as your Arweave Gateway
    
    `localStorage.setItem('flutter.arweaveGatewayUrl', '"https://ar-io.net"');`
    
    and hit `enter`.
    
    ![](blob:https://ardrive.atlassian.net/6059ebad-22ad-4f6f-bab9-f16afd414da4#media-blob-url=true&id=fc719306-15f7-462c-a02f-00dd4520c4b6&collection=contentId-635174929&contextId=635174929&height=1008&width=1792&alt=)![Screenshot\_2023-01-10\_at\_12.37.06\_PM.png](https://help.ardrive.io/hc/article_attachments/12000471362203)
    
5.  The browser is now set to retrieve data from Arweave through `ar-io.net`.
    
6.  Log in to access to Arweave & ArDrive!
    

All preview links generated by ArDrive will now reference the updated gateway URL, for example `https://ar-io.net/{txID}` (with your transaction id in place of the placeholder).

**NOTE** - To remove [ar-io.net](http://ar-io.net/ "http://ar-io.net") as the default Arweave Gateway in ArDrive, repeat step 4 with:  
`localStorage.removeItem('flutter.arweaveGatewayUrl');`
