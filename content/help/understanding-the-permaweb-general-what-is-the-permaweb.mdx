Arweave stores data permanently and sustainably over time for a single up front fee. It is a data storage protocol as a service, where data is stored on an immutable blockchain and owned by a global community, [accessible through gateways](https://ar.io/what-is-a-gateway/) that make up [the permaweb](https://ar.io/what-is-the-permaweb/).

Similar to Bitcoin and Ethereum, Arweave is [free and open source software](https://github.com/ArweaveTeam/arweave). This software is run across thousands of nodes across the world. Each node is responsible for the security of the blockweave, Arweave’s version of a blockchain.

The blockweave was purpose built for the storage and replication of data over [standard HTTP and TCP/IP.](https://docs.arweave.org/developers/server/http-api) Data on the blockweave is fault tolerant, tamper proof, and paid for - Arweave is a new way to store data, unmatched by centralized storage platforms.

The native [Arweave token (AR)](https://messari.io/asset/arweave/profile) is used to pay for permanent storage space. The [price of data](https://prices.ardrive.io) is calculated by the network and each payment is split between node operators (also known as miners) and [the endowment](https://arwiki.wiki/#/en/storage-endowment). This endowment covers the cost of storing the data for at least 200 years, modeled after declining storage costs over time.

Arweave allows for a permanent and decentralized Internet. The Arweave Permaweb is a global web of pages, and applications that live forever on Arweave. Permaweb apps use a set of open, modular and interchangeable protocols designed and developed by the community.
