What is ArConnect?

ArConnect is a Chrome browser extension that makes it easy to manage your digital wallet for ArDrive.  If you’ve ever used MetaMask before, you’ll find ArConnect similar.

ArConnect simplifies the login experience because your wallet is encrypted and safely stored within ArConnect. ArDrive can then securely access it as needed.

* * *

## Get Started

1.  Go to the [**ArConnect website**](https://arconnect.io) to download the ArConnect Chrome extension

2.  Press on the ‘Download for Chrome’ button

3.  You will be redirected to the ArConnect page of the Chrome store

4.  Once at that page press the ‘Add to Chrome’ button

5.  A pop-up will appear and please press ‘add extension’ and you will be directed to the Welcome page for ArConnect

## Download ArConnect

We are now on the “Welcome to ArConnect” page

1.  Create a password for ArConnect. This password is not the same as your password for ArDrive.

**Note:** Make sure to write down your ArConnect password as it can’t be recovered.

2\. You will be asked to either create a new wallet or load a wallet.

## Add a Wallet

-   To make a new wallet with ArConnect, press the “New Wallet” button, then copy the seedphrase and download the keyfile.

**Note:** Make sure to save your seedphrase and keyfile somewhere safe as they are not able to be recreated if lost.

-   The new wallet will be loaded into ArConnect automatically, but will have no AR.

## Log into the ArDrive using the digital wallet from ArConnect

1.  Visit the [**ArDrive login screen**](https://app.ardrive.io)

2.  Select the ArConnect option

3.  Log in with your ArDrive password
