[app.ardrive.io](https://app.ardrive.io/) connects to Arweave gateways to upload data, download data, and query indexed drive, file, and folder information.

Changing your gateway can be useful if the default gateway is temporarily unavailable, your ISP blocks access to it, or you want to use another decentralized gateway for Permaweb services.

This setting is intended for advanced users. Changing gateways can affect loading, syncing, previews, and how quickly newly uploaded data appears in the app.

# Change gateways from account settings

After you are logged in, open your account menu and go to settings.

![ArDrive account menu with settings option](/help/help-gateway-settings-1.jpg)

From settings, open the gateway settings area.

![ArDrive settings screen showing gateway settings](/help/help-gateway-settings-2.jpg)

You can update both gateway fields:

- Data Gateway: used for retrieving data and generating preview links.

- GraphQL Gateway: used for querying indexed drive, folder, and file metadata.

![ArDrive gateway settings fields](/help/help-gateway-settings-3.jpg)

# Change gateways while logging in

You can also change gateway settings from the login screen before accessing your drive.

![ArDrive login gateway settings](/help/help-gateway-login-settings.jpg)

# When to try this

Changing gateways may help if you are an advanced user troubleshooting drive syncing, file previews, or temporary gateway availability issues.

If your issue appears to be an extended outage, check the [ArDrive Discord](https://discord.com/invite/ya4hf2H) before changing settings.
