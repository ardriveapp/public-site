All of the files on your Public Drive are given a URL that can be searched and shared as any other web address.

To locate the full URL address highlight a file within your public folder.

In the details panel, select the square with a diagonal arrow - this is the public preview button. It will load any browser compatible data. This includes photos, music, movies, and PDFs.

## URLs on the Permaweb

When you click the public preview button, a new tab in your browser will open. You're viewing your data on the permaweb. The first string of numbers is a sandbox that turbo-gateway.com uses, the string of characters after turbo-gateway.com is the data item's transaction ID.

Once you have this unique URL you can share it with others, embed it on a website or bookmark it in your browser as a record of your public permaweb files.

**Warning:**  Not all file types can be previewed in the browser. If a file type cannot be previewed, it will automatically download the file to your computer.
